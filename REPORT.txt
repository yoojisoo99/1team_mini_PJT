1. 사용자 인증 및 맞춤형 프로파일링 시스템
- 회원가입 및 로그인 시스템을 구현하여 안전한 사용자 인증 흐름 구축
- 설문조사 결과를 기반으로 사용자의 투자 성향을 진단하고, 해당 데이터를 안정적인 JSON 포맷으로 분리 저장 및 연동하는 시스템 구현

2. 홈 화면(직관적인 시장 개요 대시보드) 구현
- 자체 스크래퍼 모듈을 연동하여 코스피/코스닥 상위 50~200개 종목의 실시간 등락 정보 및 거래량 표시
- Plotly와 Matplotlib 라이브러리를 활용해 주식 데이터를 시각화 차트로 렌더링
- 주식 초보자를 배려한 용어 설명 제공 및 눈 피로도를 낮추는 다크 테마 기반 UI 적용

3. 사용자 맞춤형 종목 추천 알고리즘
- 진단된 사용자 투자 성향과 개별 주식의 특성을 매칭하여 최적의 종목 추천
- 해당 종목이 왜 사용자의 성향에 부합하는지 논리적인 추천 사유를 함께 제공

4. 백그라운드 데이터 수집 및 자동화 파이프라인
- 수동 개입 없이 1시간 단위마다 최신 증시 데이터를 자동으로 스크래핑하고 갱신하는 스케줄러 파이프라인 완성
