# 주식 추천 시스템 프론트/백엔드 진행 보고서 📊

작성일자: 2026.02.24
대상 브랜치: `devbr` (로컬 저장용)

## 📌 주요 구현 내용

**1. 사용자 인증 및 맞춤형 프로파일링 시스템**
- **회원가입/로그인 시스템 구현:** 보안과 세션 관리를 적용한 안전한 사용자 인증 흐름 구축
- **투자 성향 분석 로직 개발:** 설문조사 결과를 기반으로 사용자의 투자 성향을 진단하고, 해당 데이터를 안정적인 JSON 포맷으로 분리 저장 및 연동하는 시스템 구현

**2. 홈 화면(직관적인 시장 개요 대시보드) 구현**
- **실시간 주식 정보 제공:** 자체 스크래퍼 모듈을 연동하여 코스피/코스닥 시가총액 상위 50~200개 종목의 실시간 등락 정보 및 거래량을 원활하게 표시
- **증시 데이터 시각화:** Plotly와 Matplotlib 라이브러리를 활용해 방대한 주식 데이터를 한눈에 파악할 수 있는 차트 렌더링
- **주린이 맞춤형 UI/UX:** 주식 초보자를 배려한 용어 설명 제공 및 피로도를 낮추는 다크 테마 기반의 쾌적한 인터페이스 설계

**3. 사용자 맞춤형 종목 추천 알고리즘**
- **성향별 매칭 시스템:** 진단된 사용자 투자 성향(위험 수용도)과 개별 주식의 특성(변동성, 성장성 등)을 매칭하여 최적의 포트폴리오 종목 추천
- **추천 사유 및 해석 제공:** 단순 종목 나열을 넘어, 해당 종목이 왜 사용자의 성향에 부합하는지 논리적인 사유를 함께 제공하여 신뢰도 향상

**4. 백그라운드 데이터 수집 및 자동화 파이프라인**
- **1시간 단위 자동화 스케줄러 구축:** 수동 개입 없이 일정 주기(1시간)마다 최신 증시 데이터를 자동으로 스크래핑하고 시스템을 갱신하는 파이프라인 완성
